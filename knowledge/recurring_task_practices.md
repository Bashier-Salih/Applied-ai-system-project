# Recurring Task Best Practices

## Choosing daily vs. weekly recurrence
Tasks tied to biological needs (feeding, medication, litter box cleaning, basic exercise) are almost always better modeled as "daily" recurring tasks. Tasks like nail trims, deep grooming sessions, or supply restocking are typically better modeled as "weekly" (or less frequent) recurring tasks, since doing them daily would be unnecessary or even harmful (e.g. daily nail trims).

## Keeping preferred times stable
For recurring tasks, keeping the same preferred_time across occurrences (rather than letting it shift) makes the schedule more predictable for both the owner and the pet, and makes it easier to notice when a task has been skipped.

## Avoiding recurrence overload
If an owner's available_hours are limited, having too many high-priority recurring tasks can consistently push lower-priority one-off tasks out of the schedule. When a plan repeatedly skips the same lower-priority task day after day, it's worth surfacing that pattern rather than treating each day's skip as an isolated event.

## Recurrence and life stage changes
Recurrence intervals that made sense for a puppy or kitten (e.g. frequent short feedings) often need to be revisited as the pet matures into an adult, since nutritional and exercise needs change with age.
